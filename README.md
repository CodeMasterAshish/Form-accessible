# Profile Dashboard UI

A clean and modern Profile Dashboard built using **HTML** and **CSS**. This project focuses on creating a user profile update interface with form validation states, a profile completion progress indicator, and a task checklist panel.

## Features

* Responsive profile update form
* Full Name input field
* Email validation error state
* Password and Confirm Password fields
* Eye icon for password visibility UI
* Circular Profile Completion Indicator
* Progress Checklist Section
* Modern card-based layout
* Pure HTML and CSS implementation

## Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Conic Gradient
* CSS Positioning
* Font Awesome Icons
